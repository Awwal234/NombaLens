# 🚀 NombaLens Frontend

NombaLens is an AI-powered financial insights dashboard built for Nigerian SMEs. It leverages the Nomba platform to provide intelligent business analytics, transaction monitoring, AI-generated summaries, and a financial copilot to help merchants make smarter decisions.

## ✨ Features

- 🔐 Secure Authentication
- 📊 Business Dashboard
- 🤖 AI Financial Copilot
- 📈 AI Summary & Insights
- 💳 Transaction History
- 🏦 Virtual Account Details
- 📱 Fully Responsive UI
- ⚡ Built with Vue 3 + TypeScript + Tailwind CSS

---

## 🛠 Tech Stack

- Vue 3
- TypeScript
- Vite
- Tailwind CSS
- Pinia
- Vue Router
- Socket.IO
- ApexCharts

---

## 🚀 Getting Started

Clone the repository:

```bash
git clone <repository-url>
cd nombalens
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
